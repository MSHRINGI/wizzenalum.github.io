Sum or Product

Write a program that asks the user for a number N and a choice C. And if choice...
C=1 then print sum.
C=2 then print product.
C= anyting print -1
of all integers in the range 1 to N(including)

import java.util.Scanner;
public class Solution {
public static void main(String[] args) {
Scanner scan = new Scanner(System.in);
int n = scan.nextInt();
int c = scan.nextInt();
// int n = 5, c=2;
if(c==1){
int sum = 0;
for(int i=1;i<=n;i++){
sum +=i;
}System.out.print(sum);
}else if(c==2){
int product = 1;
for (int i=1; i<=n; i++) {
product*=i;
}System.out.print(product);
}else System.out.print("-1");
}
}

Terms of AP

Write a program to print first x terms of the series 3N + 2 which are not multiples of 4.

import java.util.Scanner;
public class Main {
public static void main(String[] args) {
Scanner scan = new Scanner(System.in);
int x = scan.nextInt();
int k = 1;
for (int i=1; k<=x; i++) {
if(!((3*i+2)%4==0)){
System.out.printf("%s ",3*i+2);
k++;
}
}
}
}

Reverse of a number

Write a program to generate the reverse of a given number N. Print the corresponding reverse number.

import java.util.Scanner;
public class Main {
public static void main(String[] args) {
Scanner scan = new Scanner(System.in);
int n = scan.nextInt();
int newNumber = 0, updated = n;
do{
newNumber = updated%10 + newNumber*10;
updated = updated/10;
}while(updated >0);
System.out.print(newNumber);
}
}

sequence Check

you are given a sequance of number from input. and you have to check is this sequance is correct or not.
1. if sequance strictly increasing or decreasing then print true
2. if sequance first strictly decreasing and then start strictly increasing then also print true.. 3. any other case print false.
Note : We say that x is strictly larger than y when x > y.

import java.util.Scanner;
public class Main {
public static void main(String[] args) {
Scanner scan = new Scanner(System.in);
int i=1, n = scan.nextInt();
int preNumber = scan.nextInt(),cNumber=scan.nextInt();
for (i=3; preNumber>cNumber && i<=n; i++) {
preNumber=cNumber;
cNumber=scan.nextInt();

for (i=i;preNumber<cNumber && i <=n;i++){
preNumber=cNumber;
